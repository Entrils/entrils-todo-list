# Entrils To‑Do

Минималистичный таск‑менеджер с проектами, тегами и дедлайнами.

https://entrils-todo-list.netlify.app/ - Demo

## Возможности
- Списки задач с проектами, тегами, приоритетом и дедлайном
- Быстрая навигация: все, сегодня, предстоящие, просрочено, выполненные
- Поиск по задачам, проектам и тегам
- Экспорт/импорт JSON
- Темы оформления (зелёная, тёмная, голубая)

## Стек
- React + Vite
- CSS Modules
- LocalStorage

## Установка и запуск
```
npm install
npm run dev
```

## Экспорт/импорт
Экспорт сохраняет задачи, проекты, теги и выбранную тему в JSON‑файл. Импорт принимает файл такого же формата.
